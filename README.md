# Portal STVP - Sindicato de Vigilancia Privada

Este es el portal oficial para afiliados del STVP, desarrollado con **Streamlit**.

## Características
- Consulta de deudas por DNI.
- Formulario de afiliación digital.
- Acceso a credenciales digitales.
- Información de contacto y beneficios.

## Cómo ejecutar localmente
1. Instalar dependencias: `pip install -r requirements.txt`
2. Ejecutar: `streamlit run app.py`
